题目1：校园导航系统（图结构应用）
核心知识点：图的存储（邻接矩阵/邻接表）、最短路径算法（Dijkstra / Floyd）、图的遍历
任务要求：
1. 用图表示校园建筑与道路，顶点为建筑，边为道路及距离。
2. 实现功能：
◦ 查询任意两栋楼之间最短路径与距离
◦ 遍历校园所有建筑（深度/广度优先）
◦ 支持添加、删除建筑或道路
3. 输出路径文字描述或简易地图。

题目2：
哈夫曼编码与文件压缩（树与贪心算法）
核心知识点：二叉树、哈夫曼树、哈夫曼编码、优先队列/堆、文件操作
任务要求：
1. 统计文本文件中字符出现频率，构建哈夫曼树。
2. 生成每个字符的哈夫曼编码。
3. 实现：
◦ 对文本文件进行编码压缩
◦ 根据编码表解压缩还原原文
4. 对比压缩前后文件大小。

题目3：
给定一个静态数组，支持多次查询区间 [l, r] 中第 k 小的数，要求使用可持久化线段树实现，保证时间与空间效率。
功能要求：1. 对数组进行离散化处理。2. 构建可持久化权值线段树。3. 支持：区间第 k 小查询；区间不同元素个数查询（可选加分）4. 输出每一步操作结果，并展示版本树结构（简化版）。
难点：
• 理解可持久化思想：每次修改只复制变化路径，共享不变结构。
• 离散化 + 权值线段树结合。
• 版本控制与历史状态回溯。
四、考核点
• 线段树、前缀和思想
• 可持久化结构实现
• 时间/空间复杂度分析
• 代码健壮性与注释规范
